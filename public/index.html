Lightquark

Lightquark is an open instant messaging platform, similar to Discord and Slack.

Lightquark does not have an official client, and instead anyone can make their own or use one created by the community.

Lightquark is currently in development, and new features are being added constantly. Current features include:

  • Quarks (equivalent of servers on other platforms)
  • Channels
  • Message editing, deletion and replies
  • Custom features implemented by clients, like /me, themes and more
  • Avatars (animated ones too, with no subscription needed)
  • Authentication powered by LITauth

Aspects that may be interesting to potential client developers:

  • Easy to use API
  • Powerful tools for custom features, such as attributes on messages
  • Tools for interoperability with other clients
  • Open source, and free to use
