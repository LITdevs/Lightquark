Lightquark Developers

This page contains tools for improving interoperability with other clients

  • Client attribute search
    Implementing a new custom feature? Search for previously used clientAttributes.
    See if someone else has done it before, and enable users of both clients to use the feature. Make sure to also check the official specialAttributes.
  • Preference search
    Implementing a new setting? Search for existing preferences
    See if another client is already using it, and allow users to carry over settings to both clients.
